## 文件格式

本仓库包含多种格式的学习资料：

* PDF 格式的笔记
* Markdown (md) 格式的笔记
* 多种编程语言的代码笔记
* MarginNote 格式的笔记
* 包含重难点标注的课程 PPT
* 包含重难点标注的辅导书习题集

对于过大的文件，我们进行了分卷压缩处理。
